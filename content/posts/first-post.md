## 새로운 테마, 새로운 마음가짐

사용 툴 : **zola**, **Prism 테마**

### 마크 다운 형식으로 작성되는 블로그

* 글씨 양옆에 별 두 개를 붙이면 **굵은 글씨**가 된다.
* 글씨 양옆에 별 한 개를 붙이면 *기울임꼴*이 된다.
* 하이픈(-)이나 별(*)을 쓰고 띄어쓰기를 하면 이렇게 깔끔한 글머리 기호(리스트)가 된다.

> 인용구를 쓰고 싶을 땐 이렇게 꺾쇠(>)를 맨 앞에 달아주면 아주 예쁜 박스가 생긴다.

카테고리 기능, 테그 기능 작성자, 깃허브 페이지 연동 성공.
배너, SVG 이미지, BG 이미지 적용 성공.

### 추가 목표
* Cloudflare 적용.
* wasmtime 을 BackEnd로 사용하여 github page 에서 동적 기능 제공.

# 마크다운이란?

**마크다운(MarkDown)이란?** 존 그루버와 애런 슈워츠가 만든 HTML처럼 어려운 문서 대신 읽기 쓰기가 쉬운 문서 양식 지향 마크업 언어이다.
기본적인 확장자는 markdown을 줄인 .md 파일이고 작동원리는 마크다운 언어를 HTML 형식으로 변환하는 스크립트를 사용하게 된다.
지금 작성되는 zola로 만든 기술 blog 또한 마크다운 언어로 작성되어 있다!

# 마크다운 문법


## 1. 제목

HTML에서 처음 배우는 `<h1>` 부터 `<h6>`를 대신하는 기호

> 작성 예시
```
# h1 테그
## h2 테그
### h3 테그
#### h4 테그
##### h5 테그
###### h6 테그
```

> 실제 페이지 에서
# h1 테그
## h2 테그
### h3 테그
#### h4 테그
##### h5 테그
###### h6 테그
